# FlowSync

`FlowSync` — это веб-приложение для учета и согласования задач во внутренних IT-проектах.

Система разработана в рамках ВКР и позволяет:
- выполнять вход в систему с учетом ролей;
- создавать и редактировать задачи;
- назначать исполнителя и проверяющего;
- изменять статус задачи по этапам жизненного цикла;
- вести историю действий по задаче;
- формировать отчеты и выгружать их в `Excel` и `Word`;
- просматривать журнал аудита.

## Технологии

- `PHP`
- `MySQL`
- `Bootstrap 5`
- `JavaScript`
- `XAMPP`

## Роли пользователей

- `Менеджер` — создает и редактирует задачи, назначает сотрудников, закрывает задачи, работает с админ-панелью
- `Исполнитель` — берет задачу в работу и отправляет на проверку
- `Проверяющий` — утверждает задачу или возвращает ее на доработку

## Структура проекта

- `htdocs/` — страницы приложения
- `includes/` — серверная логика и подключение к базе данных
- `assets/` — стили и JavaScript
- `exports/` — папка для экспортируемых отчетов
- `dump.sql` — структура базы данных и тестовые данные

## Как запустить локально

1. Установить и запустить `XAMPP`
2. Скопировать проект в папку:

```text
G:\xampp\htdocs\diplom
```

3. Включить в `XAMPP Control Panel` модули:
- `Apache`
- `MySQL`

4. Открыть `phpMyAdmin`
5. Создать базу данных `diplom_tasks`
6. Импортировать файл `dump.sql`
7. Проверить настройки подключения в файле:

```text
includes/config.php
```

8. Открыть в браузере:

```text
http://localhost/diplom/htdocs/
```

## Тестовые пользователи

После импорта `dump.sql` доступны следующие учетные записи:

- `manager@example.com` / `123456`
- `executor@example.com` / `123456`
- `reviewer@example.com` / `123456`

## Что посмотреть после входа

- `Кабинет` — главная страница пользователя
- `Задачи` — список задач, создание, редактирование, смена статуса
- `Отчетность` — фильтры и экспорт отчетов
- `Админ` — пользователи, проекты, журнал аудита
